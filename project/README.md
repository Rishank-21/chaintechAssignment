# Account Management App (React)

Simple React app for an interview assignment that lets users register, log in, and view/edit their profile. Data is stored in `localStorage`.

## Features
- Register with basic validation
- Login with credential check
- Protected profile page
- Edit profile details (name, email, password)
- Logout

## Tech Stack
- React + Vite
- React Router
- Tailwind CSS
- Bootstrap (included, minimal use)

## Setup
1. Install dependencies:
```bash
npm install
```

2. Start the dev server:
```bash
npm run dev
```

3. Open the URL shown in the terminal.

## Notes
- This uses `localStorage` for demo purposes.
- Passwords are stored in plain text only for the assignment demo.
